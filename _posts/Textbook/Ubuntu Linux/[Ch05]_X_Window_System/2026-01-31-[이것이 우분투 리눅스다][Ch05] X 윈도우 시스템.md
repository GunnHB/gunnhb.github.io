## 그놈 데스크톱 환경 설정
**우분투 데스크톱<sup>Ubuntu Desktop</sup>**에서 기본으로 제공하는 X 윈도우의 오픈 데스크톱 환경인 **그놈<sup>GNOME</sup>**은 사용자의 취향에 맞게 설정할 수 있다.

- 바탕화면 변경
- 테마 변경

뿐만 아니라 GRUB의 배경에 이미지가 나오도록 변경할 수 있다.

```bash
apt -y install grub2-splashimages
```

위 명령을 실행하면 GRUB 화면에 보일 수 있는 그림들이 설치된다.

```bash
ls -l /usr/share/images/grub
```

위 명령으로 그림의 파일명을 확인해볼 수 있다. 그림은 파일 형식은 `*.tga`이다.

그 후 /etc/default/grub 파일을 편집기로 열어 `GRUB_TIMEOUT_STYLE`과 `GRUB_TIMEOUT` 행을 주석 처리한 뒤 다음 행을 추가한다.

```bash
GRUB_BACKGROUND=/usr/share/images/grub/원하는그림파일.tga
```

끝으로 `update-grub` 명령으로 설정한 내용을 적용한 뒤 재부팅하면 된다.

## X 윈도우 응용 프로그램
### 노틸러스
**노틸러스<sup>Nautilus</sup>**는 그놈 데스크톱 환경에서 제공되는 파일 관리자로 Windows의 파일 탐색기와 유사하다.

### 인터넷 응용 프로그램
인터넷과 관련된 응용 프로그램에는 웹 브라우저(Firefox), 메일 클라이언트(썬더버드), 메신저(피진), FTP 클라이언트(gFTP)가 있다. Firebox를 제외한 프로그램들은 기본으로 포함되지 않아 ***별도의 설치가 필요하다.***

- 썬더 버드<sup>Thunderbird</sup>
  - MS의 Outlook과 유사한 이메일 클라이언트

```bash
apt -y install thunderbird   # 썬더버드 설치 명령
```

- 피진<sup>Pidgin</sup>
  - AIM, Google Talk, IRC, MSN, QQ Yahoo 등과 유사한 메신저 프로그램

```bash
apt -y install pidgin   # 피진 설치 명령
```

- gFTP
  - 리눅스에서 파일을 간편하게 주고받을 수 있는 프로그램

```bash
apt -y install gftp   # gftp 설치 명령
```

### 문서 편집기/뷰어
문서 편집과 관련된 프로그램으로는 gedit, 다중 문서 뷰어, PDF 파일 전용 뷰어 정도가 있다.

- `gedit`
  - 일반적인 텍스트 편집기
  - Windows의 메모장과 유사

- `evince`
  - PDF, XPS, TIFF 등의 다중 문서를 볼 수 있는 뷰어 프로그램

- `Foxit PDF Reader`
  - PDF 파일 전용 뷰어

```bash
wget http://cnd01.foxitsoftware.com/pub/foxit/reader/desktop/linux/2.x/2.4/en_us/FoxitReader.enu.setup.2.4.4.0911.x64.run.tar.gz    # 셋업 파일 다운로드 명령
```

파일을 받은 후 다음 명령을 실행해 설치하면 된다.

```bash
tar xfz Fox*      # 압축 해제
ls Fox*
./Foxit*.run      # 설치 진행
```

### CD/DVD 레코딩 - 브라세로
우분투 24.04 LTS는 CD/DVD를 레코딩하는 툴로 **브라세로<sup>Brasero</sup>**라는 프로그램을 지원한다.

```bash
apt -y install brasero    # 브라세로 설치
brasero                   # 브라세로 실행
```

### 그래픽 프로그램
- `gimp`
  - 포토샵과 유사한 그래픽 응용 프로그램

```bash
apt -y install gimp
```

- `eog`
  - 그림 파일을 보여주는 그래픽 응용 프로그램
  - `eog` 명령을 프로그램 실행

### LibreOffice
문서 편집 프로그램에는 **LibreOffice**가 있다. MS Office와 호환성이 매우 좋기 때문에 훌륭한 대체제이다. 

|<center>MS Office</center>|<center>LibreOffice</center>|
|--|--|
|<center>워드<sup>Word</sup></center>|<center>라이터<sup>Writer</sup></center>|
|<center>엑셀<sup>Excel</sup></center>|<center>켈크<sup>Calc</sup></center>|
|<center>파워포인트<sup>Powerpoint</sup></center>|<center>임프레스<sup>Impress</sup></center>|
|<center>엑세스<sup>Access</sup></center>|<center>베이스<sup>Base</sup></center>|

```bash
snap install libreoffice    # LibreOffice 설치 명령
```

> 패키지 관리 명령으로 `apt`를 많이 사용했지만 이는 우분투 전용 명령이다. 이에 반해 `snap`은 모든 리눅스에서 공통으로 패키지를 관리하기 위해 개발되었다.
{: .prompt-tip}

### Visual Studio Code
MS의 `Visual Studio Code`는 Linux에서도 사용할 수 있다. 참고로 Snap 패키지에 포함되어 있다.

```bash
snap install --classic code     # 설치 명령
```

> `--class` 옵션은 실행되는 응용 프로그램에 리눅스의 필요한 권한을 부여한다.
{: .prompt-tip}

```bash
code --no-sandbox --user-data-dir .   # VS Code 실행
```

> `--no-sandbox` 옵션은 root가 실행할 때 필요한 옵션이며, `--user-data-dir <디렉토리>`는 VS Code에서 사용할 사용자 디렉토리를 지정하는 옵션이다.
{: .prompt-tip}
