## 시작하기 전에

[교재 공식 카페](https://cafe.naver.com/thisislinux/7818)를 통해 설치에 필요한 파일들을 미리 내려받자.

## Ubuntu Desktop
Ubuntu Desktop을 설치하기 위해 VMware에서 `Server`를 띄운 다음 [Edit virtual machine settings]를 선택한다.

![](/assets/img/post/Textbook/Ubuntu%20Linux/Ch03/Ubuntu_Desktop_001.png)

[CD/DVD]를 선택한 후 [Connected at power on]을 체크한 뒤, [Use ISO image file]에서 [ubuntu-24.04-desktop-amd64]를 선택 후 부팅한다.

![](/assets/img/post/Textbook/Ubuntu%20Linux/Ch03/Ubuntu_Desktop_002.png)

이렇게 되면 우분투를 설치할 수 있게 된다.

![](/assets/img/post/Textbook/Ubuntu%20Linux/Ch03/Ubuntu_Desktop_003.png)

설치를 진행하다보면 계정 설정 화면이 나온다. 교재와 똑같이 진행했다.

![](/assets/img/post/Textbook/Ubuntu%20Linux/Ch03/Ubuntu_Desktop_004.png)

여러 설정들을 해주면 마침내 우분투 설치가 완료된다😊

## Ubuntu Server
다음으로 Ubuntu Server를 설치한다. 마찬가지로 VMWare에서 `Server(B)`를 띄운 후 [Edit virtual machine settings]를 선택한다.

![](/assets/img/post/Textbook/Ubuntu%20Linux/Ch03/Ubuntu_Server_001.png)

[CD/DVD]를 선택한 후 [Connected at power on]을 체크한 뒤, [Use ISO image file]에서 [ubuntu-24.04-live-server-amd64]를 선택 후 부팅한다.

> Ubuntu Desktop과 달리 Ubuntu Server는 **X 윈도우**를 사용하지 않기 때문에 텍스트 모드로만 사용된다.
{: .prompt-info}

![](/assets/img/post/Textbook/Ubuntu%20Linux/Ch03/Ubuntu_Server_002.png)

Ubuntu Server 역시 계정을 설정해야 한다. 이것 역시 교재와 똑같이 했다.

![](/assets/img/post/Textbook/Ubuntu%20Linux/Ch03/Ubuntu_Server_003.png)

설치가 완료되고 재부팅 후 로그인을 한다. 이러면 명령어를 통해 리눅스를 관리할 수 있게 된다.

## Ubuntu Budgie
다음으로 Ubuntu Budgie를 설치한다. VMWare에서 `Client`를 띄운 후 [Edit virtual machine settings]를 선택한다.

![](/assets/img/post/Textbook/Ubuntu%20Linux/Ch03/Ubuntu_Budgie_001.png)

[CD/DVD]를 선택한 후 [Connected at power on]을 체크한 뒤, [Use ISO image file]에서 [ubuntu-budgie-24.04.3-desktop-amd64]를 선택 후 부팅한다.

> Ubuntu Desktop은 GNOME이라는 GUI 환경을 사용하지만, Ubuntu Budgie는 GNOME 대신 Budgie Desktop 환경이 GUI로 제공된다.
{: .prompt-info}

이후 Ubuntu Desktop과 유사하게 설치를 진행하면 된다.

![](/assets/img/post/Textbook/Ubuntu%20Linux/Ch03/Ubuntu_Budgie_002.png)
_Ubuntu Budgie 설치가 완료된 모습_
