## 호스트 이름 확인하기
호스트 이름을 변경하기 전에 호스트 이름을 확인하는 명령어는 다음과 같다.

```bash
# 호스트 이름만 확인
hostname

# 상세 정보 확인
hostnamectl
```

## 호스트 이름 변경하기
호스트 이름을 변경하기 위해 다음 명령어를 입력한다.

```bash
sudo hostnamectl set-hostname 변경할 이름
```

이는 시스템에서 사용하는 이름을 변경하는 작업이다. 네트워크 통신을 위해 `/etc/hosts` 파일에서의 호스트 이름을 똑같이 수정해주면 된다.

```bash
sudo vi /etc/hosts
```

![](/assets/img/post/Textbook/Ubuntu%20Linux/linux_hostname.png)

그 후 재부팅을 해주면 호스트 이름이 바뀐 것을 확인할 수 있다.
