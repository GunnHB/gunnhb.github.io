## EC2
**EC2**<sup>Elastic Compute Clude</sup>는 AWS에서 제공하는 가상 서버이다. 아마존의 거대한 데이터 센터 컴퓨터 자원 중 일부를 내 서버처럼 빌려 쓰는 서비스라고 생각하면 된다.

## EC2의 구성
1. 인스턴스<sup>Instance</sup>
  - 클라우드 상에서 띄운 가상 서버 한 대

2. AMI<sup>Amazon Machine Image</sup>
  - 인스턴스를 만들 때 뼈대가 되는 운영체제<sup>OS</sup> 템플릿

3. 인스턴스 유형<sup>Instance Type</sup>
  - 서버의 하드웨어 사양

4. EBS<sup>Elastic Block Store</sup>
  - EC2 인스턴스에 꽂아 사용하는 가상 하드디스크

## 부트스트래핑
**부트스트래핑**<sup>bootstrapping</sup>은 EC2 인스턴스가 처음 생성되어 부팅될 때 미리 작성해 둔 스크립트나 명령어를 자동으로 실행하도록 만드는 과정이다.

AWS에서 EC2 인스턴스를 생성할 때 **사용자 데이터**<sup>User Data</sup>라는 항목에 스크립트를 입력하여 부트스트래핑을 구현한다. 

예를 들어 EC2가 켜지자마자 자동으로 웹 서버를 설치하고 실행하게 만들고 싶다면 다음과 같이 입력할 수 있다.

```bash
#!/bin/bash
# 패키지 업데이트
apt-get update -y
# 아파치 웹 서버 설치
apt-get install -y apache2
# 아파치 서비스 시작 및 부팅 시 자동 실행 설정
systemctl start apache2
systemctl enable apache2
# 간단한 웹 페이지 생성
echo "<h1>Hello from EC2 Bootstrapping!</h1>" > /var/www/html/index.html
```

이렇게 하면 관리자가 SSH로 서버에 일일이 접속해서 명령어를 칠 필요없이 인스턴스가 켜지고 몇 분 뒤에 곧바로 웹 서비스가 준비된 상태가 된다.
