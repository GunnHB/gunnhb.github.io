## AWS에 접근하는 방법
AWS에 접근하는 방법에는 세 가지가 있다.

1. AWS 관리 콘솔
  - 웹을 통해 접속하는 시각적인 화면
  - 비밀번호 + MFA

2. AWS CLI
  - 터미널이나 명령 프롬프트 창
  - Access Key

3. AWS SDK
  - 개발 환경에서 AWS 서비스 기능을 호출
  - Access Key

Access Key를 생성하면 Access key ID와 Secret access key를 발급받을 수 있다. Secret access key는 잃어버리면 다시 찾을 수 없으니 반드시 다른 곳에 저장해두어야 한다.

![](/assets/img/post/Amazon%20Cloud/IAM/Access_key.png)

> Access Key는 개인이 가지는 것이므로 다른 사용자들과 공유해서는 안된다.
