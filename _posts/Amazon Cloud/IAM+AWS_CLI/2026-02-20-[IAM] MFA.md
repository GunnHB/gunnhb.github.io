## MFA
**MFA**<sup>Multi Factor Authemtication</sup>는 AWS에서 권장하는 다중 인증 방식이다. 이는 비밀번호와 소유한 보장 장치의 조합으로 사용된다. 만약 해커가 특정 사용자의 비밀번호가 알고 있더라도 물리적 장치까지 확보하지 않는 이상 계정에 접근하기는 어렵다.

## MFA 장치
- Virtual MFA device
  - Google Authenticator
  - Authy

- Univeral 2nd Factor Security Key
  - YubiKey

- Hardware Key Fob MFA Device
  - Provided by Gemalto

- Hardware Key Fob MFA Device for AWS GovCloud
  - Provided by SurePassID
