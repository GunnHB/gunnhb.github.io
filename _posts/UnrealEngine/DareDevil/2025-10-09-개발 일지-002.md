## 🤔 작업 내용

```c++
UCLASS()
class DAREDEVIL_API ADareDevilBaseGameMode : public AGameModeBase
{
	GENERATED_BODY()
	
};
```

그 후 프로젝트 세팅에서 해당 게임모드로 교체해줬다.

다음으로 해당 프로젝트에서는 **GameplayAbilitySystem(GAS)**를 사용할 것이기에 플러그인도 설치했다.
